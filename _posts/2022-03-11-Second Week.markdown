# 2주차 배운 내용 복습
## 깃 학습 비주얼 코드 사용법


1. git 가입
2. 비주얼코드로 마크다운 언어를 통해 코드 작성
3. 터미널창에서 지정파일(my-awesome-site)로 이동후 
4. git add . 명령어를 통해 추가\
전부다 추가되기 때문에 따로 추가하기 싫은 파일이 있다면 자동적으로 되지 않게 변경
5. git commit -m "메세지" 명렁어를 통해 내가 업데이트한 버젼 및 뭐를 업데이트햇는지 확인
6. git push 업데이트한 내용을 gitHub에 업데이트
